SuperPlatformer

Description du jeu :

SuperPlatformer est un jeu de plateforme créé par le groupe ACHC.
Celui-ci a été créé dans le cadre du projet de S2 de l'école d'ingénieur EPITA.
Il reprend le style de grands jeux comme Super Mario.
